# Campus Lost & Found Management System
## Performance Optimization
The system was optimized to ensure good performance for campus users. 

Two optimizations were implemented:
1. **Deferred JavaScript Loading**  
   The main JavaScript file is loaded using the `defer` attribute to prevent render-blocking and allow the HTML content to load first.

2. **Lightweight Interface Design**  
   The application avoids large images and unnecessary assets, keeping the interface simple and fast to load.

These optimizations improved the Lighthouse performance score of the application.
